# Quizz — Juego de preguntas

Mini aplicación React (Create React App) para practicar preguntas de cultura general.

## Cómo ejecutar (local)
```bash
npm install
npm start
```

## Estructura recomendada
```
/ (root)
├─ public/
├─ src/
│  ├─ components/
│  ├─ data/
│  └─ styles/
├─ .gitignore
├─ package.json
├─ README.md
└─ LICENSE
```
