Mini aplicación React (Create React App) para practicar preguntas de cultura general.

## Cómo ejecutar (local)
```bash
npm install
npm start
```

## Estructura recomendada
```
/ (root)
├─ public/
├─ src/
│  ├─ components/
│  ├─ data/
│  └─ styles/
├─ .gitignore
├─ package.json
├─ README.md
└─ LICENSE
```

## Deploy sugerido
- GitHub Pages (usando build -> gh-pages) o Vercel/Netlify.
- Incluye un workflow de GitHub Actions en `.github/workflows/ci-pages.yml` para construir y publicar.
